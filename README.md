## **Getting Started**

### Prerequisites
- Python 3.8+
- Node.js 16+
- PostgreSQL (managed via Supabase)
- `pip`, `npm`/`yarn`

### Installation
(Only for currently implemented portions)
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/kimmmmchidjango.git
   cd kimmmmchidjango

2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt

4. Set up .env files (Backend: kimmmmchidjango/.env):
   ```
   SECRET_KEY=your-secret-key
   DEBUG=True
   DB_NAME=your-supabase-db-name
   DB_USER=your-supabase-user
   DB_PASSWORD=your-supabase-password
   DB_HOST=your-supabase-host
   DB_PORT=5432

5. Run Django server:
   ```bash
   python manage.py migrate
   python manage.py runserver
